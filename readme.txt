Dribbble Portfolio Shots is a Wordpress Plugin for displaying shortcode.

== Description ==

Dribbble becoming popular day by day for designers and designers submit their portfolio items on it. We can understand the importance of Portfolio and Dribbble Portfolio Shots Shortcode will allow developer to put designer portfolio on their website.

Few Features you will love

*   Set default values to give you easy start
*   Responsive in Nature
*   Available 2 column, 4 column, 6 column grid
*   Shortcode can be put on theme easily

== Installation ==

Installing this plugin is same like other wordpress plugins

1. Upload `dribbble-portfolio-shots` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place <?php echo do_shortcode("[dribbble_shots]"); ?> in your themes

You can install the zip file as well

Available Shortcode

1. Put your own dribbble username
[dribbble_shots id="YOUR_DRIBBBLE_NAME"]

2. for two column grid
[dribbble_shots id="YOUR_DRIBBBLE_NAME" column=2]

four column is default so doesn't need to mention
[dribbble_shots id="YOUR_DRIBBBLE_NAME" column=4]

Six column -
[dribbble_shots id="YOUR_DRIBBBLE_NAME" column=6]
